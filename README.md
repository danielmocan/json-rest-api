# json-rest-api
Fake json rest API with node

**WIP - Work in progress**

## Usage

**Clone the repo**
```
git clone https://github.com/danielmocan/json-rest-api.git
```
**Run project**
```
npm start
```
server will listen for requests on port 3000


### Routes

```
GET    /posts
GET    /posts/1
POST   /posts
PUT    /posts/1
DELETE /posts/1
```
